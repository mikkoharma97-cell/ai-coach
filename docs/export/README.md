# Siirtopaketti — indeksi

Tämä kansio on **vienti toiseen ohjelmaan / dokumenttiin / toiselle agentille**. Se on generoitu/koottu reposta; päivitä tarvittaessa uudelleen.

## Mitä täällä on

| Polku | Sisältö |
|--------|---------|
| `ai-coach/` | AI Coach -sovelluksen kattava paketti (tuote, säännöt, stack, reitit, lib-kartta, tilakone, docs-luettelo) |
| `external/` | **Erilliset** tilannekuvat muista projekteista: `SITEOS.md`, `siteos-reality-check-STANDALONE.md`, `ai-coach-safety-backups.md` |

Aloita: `ai-coach/00-INDEX.md`.

## Rajoitteet (rehellinen)

- **En sisällä:** tuotantosalaisuuksia, käyttäjien dataa, analytics-tilejä, App Store -konffeja kokonaisuudessaan.
- **Sisältää:** repossa olevat tiedostot, Cursor-säännöt tästä reposta, globaalin `closing-suggestions` -muistin kopion (jos erosi).
- **Muut projektit:** voin lukea levyltä polkuja, jotka ovat käytettävissä (Cursor-työtila voi rajoittaa — tässä passissa luettiin `/Users/mikkoharma/...`). Laajempi vienti = lisää projektit työtilaan tai anna polut.

## PDF

Tämä kansio on **Markdown**. PDF: `pandoc docs/export/README.md -o export.pdf` tai Tulosta → PDF editorista. Useampaan tiedostoon: yhdistä ensin tai tulosta kansion tiedostot erikseen.

## Päivitys

Kun koodi muuttuu, aja uusi vienti tai pyydä agenttia päivittämään nämä tiedostot.
