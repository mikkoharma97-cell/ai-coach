# Store screenshots (placeholder)

Tähän kansioon voi kopioida julkaisukanavien kuvakaappaukset (esim. 1290×2796, 1242×2688).

Generointi repossa: `npm run screenshots` → tulostuu juureen `screenshots/` (ei tässä kansiossa automaattisesti).

Konfiguraatio: `src/config/appInfo.ts` (`APP_SCREENSHOTS_PUBLIC_DIR`).
